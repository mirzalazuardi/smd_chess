## 1. Cara Mendaftar

### Sebelum Mendaftar, Siapkan:

- **Bukti transfer pembayaran** (foto/screenshot, format JPG/PNG, maksimal 5MB)
- **Nomor WhatsApp** yang aktif
- **Rating catur** (opsional, jika punya)

### Langkah-langkah:

1. Buka halaman pendaftaran: `https://[domain]/daftar`
2. Pilih turnamen yang ingin diikuti (jika ada lebih dari satu)
3. Isi data diri:

| Field | Keterangan | Contoh |
|-------|-----------|--------|
| **Nama Lengkap** | Nama sesuai KTP/kartu pelajar | Budi Santoso |
| **Email** | Email aktif untuk konfirmasi | budi@email.com |
| **Status** | Pilih **Pelajar** atau **Umum** | Umum |
| **Nama Sekolah** | Wajib diisi jika status Pelajar | SMA Negeri 1 Sumedang |
| **Nomor WhatsApp** | Hanya angka, 10-15 digit | 081234567890 |
| **Rating Catur** | Opsional. Kosongkan jika tidak punya | 1500 |
| **Bukti Transfer** | Upload foto/screenshot bukti pembayaran | `transfer.jpg` |

4. Periksa kembali semua data
5. Klik tombol **Daftar**

### Jika Berhasil:

Anda akan melihat halaman **Pendaftaran Berhasil** yang menampilkan:

```
ID Pendaftaran Anda:

    CATUR2026-XXX
```

> **PENTING: Simpan ID Pendaftaran ini!** ID ini digunakan untuk mengecek status pendaftaran Anda.

### Jika Gagal:

| Pesan Error | Solusi |
|-------------|--------|
| "Email sudah terdaftar" | Email yang sama tidak bisa mendaftar dua kali di turnamen yang sama |
| "File harus berupa gambar" | Pastikan bukti transfer dalam format JPG/PNG |
| "Ukuran file maksimal 5MB" | Kompres atau resize gambar bukti transfer |
| "Kode turnamen tidak ditemukan" | Pastikan memilih turnamen yang benar |
| "Pendaftaran sudah ditutup" | Turnamen sudah dimulai, tidak bisa mendaftar lagi |

## 2. Setelah Mendaftar

### Proses Verifikasi:

1. Setelah mendaftar, status pembayaran Anda adalah **Belum Lunas**
2. Tim admin akan memverifikasi bukti transfer Anda dalam **1x24 jam**
3. Setelah diverifikasi, status berubah menjadi **Lunas**
4. Hanya peserta dengan status **Lunas** yang akan masuk ke dalam pairing

### Cara Cek Status:

Saat ini pengecekan status dilakukan dengan menghubungi panitia melalui WhatsApp. Fitur pengecekan mandiri akan hadir di update berikutnya.

## 3. Melihat Jadwal Pertandingan

Setelah pairing digenerate oleh admin, Anda bisa melihat jadwal pertandingan:

1. Buka: `https://[domain]/jadwal/[kode-turnamen]`

   Contoh: `https://[domain]/jadwal/sumedang_open_2026`

2. Halaman akan menampilkan semua ronde dengan pasangan pertandingan

**Cara membaca jadwal:**

```
Ronde 1                              Selesai

    Budi Santoso        1 - 0        Ani Rahmawati
    Caca Putri          ½ - ½        Deni Kurniawan
    Eko Prasetyo        0 - 1        Fani Gunawan
```

| Format | Artinya |
|--------|---------|
| **1 - 0** | Pemain kiri (putih) menang |
| **½ - ½** | Remis / seri |
| **0 - 1** | Pemain kanan (hitam) menang |
| **vs** | Pertandingan belum dimulai |

**Auto-refresh:** Halaman akan otomatis refresh setiap 30 detik. Anda juga bisa klik tombol **Refresh** untuk update manual.

## 4. Melihat Klasemen

1. Buka: `https://[domain]/klasemen/[kode-turnamen]`

   Contoh: `https://[domain]/klasemen/sumedang_open_2026`

2. Halaman menampilkan peringkat semua peserta

**Cara membaca klasemen:**

| # | Nama | Poin | M | S | K | BH |
|---|------|------|---|---|---|---|
| 1 | Budi S. | 3.0 | 3 | 0 | 0 | 5.0 |
| 2 | Ani R. | 2.5 | 2 | 1 | 0 | 4.5 |

| Kolom | Artinya |
|-------|---------|
| **#** | Peringkat |
| **Poin** | Total poin (Menang = 1, Remis = 0.5, Kalah = 0) |
| **M** | Jumlah menang |
| **S** | Jumlah seri |
| **K** | Jumlah kalah |
| **BH** | Buchholz — tie-breaker: total poin lawan yang dihadapi |

**Urutan peringkat:** Poin tertinggi → Buchholz tertinggi → Rating tertinggi

> 3 besar ditandai dengan latar kuning.

**Auto-refresh:** Halaman klasemen juga otomatis refresh setiap 30 detik.

## 5. Pertanyaan Umum (FAQ)

### Pendaftaran

**Q: Saya lupa menyimpan ID Pendaftaran, bagaimana?**
A: Hubungi panitia melalui WhatsApp dengan menyebutkan nama lengkap dan turnamen yang diikuti.

**Q: Bisakah saya mendaftar lebih dari satu turnamen?**
A: Ya, dengan email yang sama Anda bisa mendaftar di turnamen yang berbeda.

**Q: Saya salah mengisi data, bagaimana cara mengubahnya?**
A: Hubungi panitia untuk perbaikan data. Fitur edit mandiri akan hadir di update berikutnya.

**Q: Apakah rating catur wajib diisi?**
A: Tidak wajib. Jika Anda tidak memiliki rating resmi, kosongkan saja.

### Pembayaran

**Q: Berapa lama verifikasi pembayaran?**
A: Maksimal 1x24 jam setelah mendaftar.

**Q: Bagaimana jika pembayaran saya ditolak?**
A: Status akan tetap "Belum Lunas". Silakan hubungi panitia untuk konfirmasi.

### Pertandingan

**Q: Kapan jadwal pertandingan muncul?**
A: Setelah admin melakukan "Generate Pairing". Biasanya H-1 atau hari-H pertandingan.

**Q: Apa itu Buchholz?**
A: Tie-breaker jika poin sama. Dihitung dari total poin semua lawan yang pernah dihadapi. Semakin tinggi Buchholz, semakin kuat lawan yang dihadapi.

**Q: Bagaimana jika saya dapat bye?**
A: Bye berarti Anda mendapat 1 poin tanpa bertanding. Ini terjadi jika jumlah peserta ganjil. Bye diberikan ke peserta dengan peringkat terendah.

**Q: Bagaimana cara mengetahui warna (putih/hitam) saya?**
A: Nama di sebelah kiri adalah pemain **putih**, sebelah kanan adalah pemain **hitam**.
